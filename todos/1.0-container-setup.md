# TODO 1.0: OpenClaw + Podman Container Setup

## Overview

Set up a sandboxed container running OpenClaw that can:
- Run autonomously in a container
- Pull code from git at startup
- Commit changes back to git
- Spin up infra (Airflow, Kafka) inside container via Podman
- Connect to Discord for messaging

## Notes

- Use Podman instead of Docker for rootless container spawning
- Git is the source of truth - not volumes
- Container is ephemeral - all state in git
- Discord is the interface - not terminal
